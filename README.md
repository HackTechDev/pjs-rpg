Phaser JS 3 RPG
===============


Installation du serveur
=======================

- Installation du module du serveur :

$ npm init -y
$ npm install lite-server 


- Configuration :

Ouvrir : package.json

Chercher :

"test": "echo \"Error: no test specified\" && exit 1"

Remplacer par :
    "test": "echo \"Error: no test specified\" && exit 1", 
    "start": "lite-server"


- Lancement du serveur :

npm start 


- Adresse internet :

http://localhost:3000
